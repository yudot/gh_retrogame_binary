GRAPHIC MODE
  • NORMAL: 320x240. Original resolution, fast

DISPLAY
Switches between window and full screen modes. Can be used by pressing ALT + ENTER.

VSYNC
Vertical synchronization. To use it you must have your monitor set to 60hz, otherwise the game will run faster than usual.
