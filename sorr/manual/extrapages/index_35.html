ADDITIONAL GAME CONTROLS
ALT + X - Close game (creates savestate)
ALT + ENTER - Switch between window/fullscreen
ENTER - General OK key
ESC - General cancel key
START or ESC - Pause game

GRAPHIC FILTERS
F1 - Original mode (320x240)
F2 - 2xScale (640x480)
F3 - HQ2X (640x480)
F4 - Scanlines (640x480)
F5 - 2xNormal (640x480)
CONTROLES ADICIONALES
ALT + X - Abandonar juego (crea un savestate)
ALT + ENTER - Cambia entre ventana y pantalla completa
ENTER - Tecla OK general
ESC - Tecla cancelar general
START o ESC - Pausar partida

FILTROS GRÁFICOS
F1 - Modo original (320x240)
F2 - 2xScale (640x480)
F3 - HQ2X (640x480)
F4 - Scanlines (640x480)
