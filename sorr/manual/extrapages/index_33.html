• The icon next to Player 1 shows the gamepad number being configured. If there are 2 gamepads connected, the second one will always be assigned to Player 2
  • When the game starts for the first time, checks are made and the first gamepad is configured. If there's none connected an empty dialog will be shown as icon
  • Once keyboard or gamepad are chosen the player can navigate through the mapping options. On-screen gamepad buttons will be lit when assigning keys/buttons
The controls map is made of the following buttons:
A, B, C, X, Y, Z, L, R and START

MENU CONTROLS
C - OK
B - Cancel
  • El icono junto a Jugador 1 muestra el número de mando configurado. Si hay 2 mandos conectados, el segundo siempre se asignará al Jugador 2.
  • Al iniciar el juego por primera vez se comprueba y configura automáticamente el primer mando. Si no hay ninguno conectado se muestra un diálogo vacío como icono
  • Una vez elegido Teclado o Mando el jugador puede moverse por las opciones de mapeado. Los botones del mando en pantalla se iluminarán al asignar teclas/botones
