AUTO SAVE
There's a checkpoint at the start of every stage. The auto save function is only activated in one of the following situations:
  • Closing the game by pressing ALT + X
  • Closing the game using the close button of the window
  • A forced game shutdown
The game state will not be saved when a game over takes place or when ending a game through the pause menu.
AUTO GUARDADO
Hay un punto de control al comienzo de cada fase. La función de autoguardado sólo se activa en una de las siguientes situaciones:
  • Al cerrar el juego presionando ALT + X
  • Al cerrar el juego presionando sobre el botón de cierre de la ventana
  • Al sufrir el juego un cierre forzado
El estado del juego no se guardará cuando tenga lugar un game over o al salir de una partida mediante el menú de pausa.
