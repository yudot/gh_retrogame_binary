COMBO TYPE
Sets the character behaviour upon pressing the B button repeatedly:
  • SOR 2: Executes the complete sequence only if they connect with an enemy, destructible object or another player character
  • SOR 3: Executes the complete sequence under any circumstance
JUMP TYPE
Sets the character behaviour upon jumping:
  • SOR 1: Allow changing both the path and the facing of a jump
  • SOR 2: Don't allow changing neither the path nor the facing of a jump
  • SOR 3: Allow changing the path but not the facing of a jump
TIPO DE COMBO
Define el comportamiento al pulsar repetidamente el botón B:
  • SOR 2: Sólo ejecuta la secuencia de golpes completa si éstos conectan con un enemigo, objeto destruible u otro jugador
  • SOR 3: Permite realizar la secuencia de golpes completa en cualquier circunstancia
TIPO DE SALTO
Define el comportamiento de los personajes al saltar:
  • SOR 1: Permite modificar tanto la trayectoria como la orientación de un salto
  • SOR 2: No permite modificar la trayectoria ni la orientación de un salto
  • SOR 3: Permite modificar la trayectoria pero no la orientación de un salto
