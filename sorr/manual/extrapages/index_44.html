NAME EDITOR


A mouse is required to use the name editor.
  • Click on an enemy box to select him/her
  • Click on a text field to edit a name
EDITOR DE NOMBRES


Se necesita un ratón para utilizar el editor de nombres.
  • Pulsa un recuadro de enemigo para seleccionarlo
  • Pulsa un recuadro de texto para editar el nombre
